Sewing Pattern Project

Individual Project

Fall 2025 - present

This personal project is a creative fusion of sustainability, fashion, and technology. The goal is to design and build an interactive interface where users can upload an image of an old or unused piece of clothing (e.g., a shirt, dress, or jeans), and receive personalized sewing pattern suggestions for upcycling it into a new, wearable item.

bolt

By combining image recognition and pattern-matching logic that is derived from fitting shapes within a larger shape, the tool aims to empower users—especially beginners and DIY enthusiasts—to give their garments a second life rather than discarding them. The final output could include links to downloadable patterns, estimated fabric yield, and styling suggestions based on the original garment. The crux of this project is connecting all of the pieces of this process, from image extraction, mesuring the relative size, finding the best pattern given the clothing item, and presenting it in a user-friendly manner.

bolt

While this project is still in the initial stages, I am excited about combining all of these different elements to create a seamless process, promoting sustainability on an individual level. To see what stage the project is at, take a look at my GitHub!
